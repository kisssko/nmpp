template<class T>
class nmmtrpack< T >

определяет классы, предназначенные для контроля переполнения при реализации библиотеки на PC.

–реализации библиотеки на PC производит контроль переполнения с выдачей сообщения об ошибке пользователю библиотеки. Для этой цели определены шаблонные классы для вектора, матрицы и скалярного числа, позволяющие производить базовые операции над их элементами.  класс матриц.
